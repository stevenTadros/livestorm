## Welcome on livestormGame ##


### Prerequisite : ###

**This App need ruby 2.6.5**

### Getting Started : ###

**After cloning the repository and once in the "livestorm" directory**
**You can set your environment and launch the game by different way**

**Rbenv exemple:**
```
 rbenv local 2.6.5
 cd livestorm
 bundle
 rake install
 cd ..
 ruby run.rb
```
